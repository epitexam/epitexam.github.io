## Les bases d’un home-server personnel
<br/>

<br/>
### 1.1 Objectifs principaux
<br/>

Centraliser (médias, backups, apps), sécuriser (accès contrôlé, chiffrement), automatiser (tâches récurrentes).  

<br/>
### 1.2 Contenu et services essentiels
<br/>

Stockage fichiers, médias (Jellyfin/Plex), backups, VPN, monitoring, IoT... Liste minimale viable.  

<br/>
### 1.3 Performance et consommation
<br/>

<br/>

Accès sécurisé depuis l'extérieur (VPN, reverse proxy).
